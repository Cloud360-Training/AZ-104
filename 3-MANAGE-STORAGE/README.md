# Module 3: Manage Storage (15-20%)

This module covers Azure Storage services, including storage accounts, blob storage, file shares, and storage security configurations.

## 📖 Microsoft Official Documentation & Learn

### 1. Manage Storage Accounts
* **Microsoft Learn Path**: [Manage Azure Storage accounts](https://learn.microsoft.com/en-us/training/modules/create-azure-storage-account/)
* **Key Topics**:
  * Create and configure storage accounts
  * Configure storage account settings
  * Manage storage account access keys
  * Configure storage account networking

### 2. Manage Data in Azure Storage
* **Microsoft Learn Path**: [Manage data in Azure Storage](https://learn.microsoft.com/en-us/training/modules/manage-azure-storage/)
* **Key Topics**:
  * Manage blob storage
  * Configure blob lifecycle management
  * Implement blob snapshots and versioning
  * Configure storage tiers (hot, cool, archive)

### 3. Configure Azure Files and Blob Storage
* **Microsoft Learn Path**: [Configure Azure Files and Blob Storage](https://learn.microsoft.com/en-us/training/modules/configure-blob-storage/)
* **Key Topics**:
  * Create and manage file shares
  * Configure Azure File Sync
  * Implement blob access tiers
  * Configure blob replication

### 4. Manage Storage Security
* **Microsoft Learn Path**: [Manage storage security](https://learn.microsoft.com/en-us/training/modules/secure-azure-storage-account/)
* **Key Topics**:
  * Configure storage account firewalls and virtual networks
  * Implement shared access signatures (SAS)
  * Configure encryption at rest and in transit
  * Manage storage account access policies
