# Hands-On Labs & Sandbox Exercises

Practical experience is essential for the AZ-104 exam, which includes performance-based tasks. This section provides guidance on accessing lab environments and completing key exercises.

## 🧪 Microsoft Learn Sandbox

The Microsoft Learn Sandbox provides a free, temporary Azure environment where you can practice deploying and managing resources without needing a credit card.

### How to Access the Sandbox
1. Navigate to any Microsoft Learn module that includes a hands-on exercise.
2. Sign in with your Microsoft account.
3. Click the **Activate sandbox** button.
4. Review the permissions and accept.
5. You will have a set amount of time (usually 1-4 hours) to use the sandbox environment.

## 📝 Recommended Lab Exercises

These exercises align with the AZ-104 exam objectives and provide practical experience with core Azure administration tasks.

### Lab 1: Create a Virtual Machine
* **Objective**: Deploy a Windows or Linux virtual machine using the Azure portal.
* **Microsoft Learn Guide**: [Exercise - Create a Windows virtual machine](https://learn.microsoft.com/en-us/training/modules/create-windows-virtual-machine-in-azure/)
* **Key Skills**: Selecting VM sizes, configuring networking, connecting via RDP/SSH.

### Lab 2: Configure VM Networking
* **Objective**: Configure network interfaces, IP addressing, and network security groups for VMs.
* **Microsoft Learn Guide**: [Exercise - Configure VM networking](https://learn.microsoft.com/en-us/training/modules/configure-network-settings-virtual-machines/)
* **Key Skills**: Network interface configuration, IP addressing, NSG rules.

### Lab 3: Deploy a Container Instance
* **Objective**: Deploy a containerized application using Azure Container Instances.
* **Microsoft Learn Guide**: [Exercise - Deploy containers using ACI](https://learn.microsoft.com/en-us/training/modules/run-docker-containers-aci/)
* **Key Skills**: Container image deployment, port configuration, environment variables.

### Lab 4: Create a Storage Account
* **Objective**: Provision an Azure Storage account and configure blob storage.
* **Microsoft Learn Guide**: [Exercise - Create a storage account](https://learn.microsoft.com/en-us/training/modules/create-azure-storage-account/)
* **Key Skills**: Storage account creation, tier configuration, access key management.

### Lab 5: Configure Blob Storage
* **Objective**: Configure blob storage tiers, lifecycle policies, and access controls.
* **Microsoft Learn Guide**: [Exercise - Configure blob storage](https://learn.microsoft.com/en-us/training/modules/configure-blob-storage/)
* **Key Skills**: Lifecycle management, access tiers, blob snapshots.

### Lab 6: Create a Virtual Network
* **Objective**: Create a virtual network with subnets and configure DNS.
* **Microsoft Learn Guide**: [Exercise - Create a virtual network](https://learn.microsoft.com/en-us/training/modules/design-implement-virtual-network/)
* **Key Skills**: VNet creation, subnet configuration, DNS settings.

### Lab 7: Configure Network Security Groups
* **Objective**: Create and manage network security groups with inbound and outbound rules.
* **Microsoft Learn Guide**: [Exercise - Configure NSGs](https://learn.microsoft.com/en-us/training/modules/secure-network-connectivity-azure/)
* **Key Skills**: NSG rule creation, traffic filtering, security best practices.

### Lab 8: Configure Load Balancing
* **Objective**: Set up Azure Load Balancer with backend pools and health probes.
* **Microsoft Learn Guide**: [Exercise - Configure load balancing](https://learn.microsoft.com/en-us/training/modules/load-balance-web-traffic-with-application-gateway/)
* **Key Skills**: Load balancer configuration, health probes, traffic distribution.

### Lab 9: Configure Azure Monitor
* **Objective**: Set up monitoring, alerts, and log analytics for Azure resources.
* **Microsoft Learn Guide**: [Exercise - Configure Azure Monitor](https://learn.microsoft.com/en-us/training/modules/monitor-azure-resources/)
* **Key Skills**: Metric collection, alert creation, log analysis.

### Lab 10: Implement Azure Backup
* **Objective**: Configure backup policies and perform backup/restore operations.
* **Microsoft Learn Guide**: [Exercise - Implement Azure Backup](https://learn.microsoft.com/en-us/training/modules/protect-virtual-machines-with-azure-backup/)
* **Key Skills**: Backup vault creation, policy configuration, restore operations.
