# AZ-104 Microsoft Azure Administrator: Training

Welcome to the **AZ-104 Microsoft Azure Administrator Ninja Training**! This repository is your comprehensive resource for preparing for the AZ-104 certification exam, one of Microsoft's most sought-after Azure certifications.

**Maintained by [Cloud360 Training](https://cloud360.co)** - Your trusted partner for Azure certification preparation.

Following the Microsoft Ninja Training format, this guide takes you from foundational concepts to advanced Azure administration through a carefully structured learning path.

## 🎯 Training Philosophy

This training program prioritizes resources in the following order to ensure you get the most accurate and effective preparation:
1. **Microsoft Official Documentation & Learn Modules** (Highest Priority)
2. **Cloud360 Training Videos & Resources**
3. **Hands-on Labs & Sandbox Exercises**
4. **Community Resources & Practice Tests**

## 📚 Course Structure

The training is organized into five modules that directly map to the official AZ-104 exam skills measured.

### [Module 1: Manage Azure Identities and Governance (20-25%)](./1-MANAGE-IDENTITIES/README.md)
* Manage Microsoft Entra users and groups
* Manage access to Azure resources
* Manage Azure subscriptions and governance

### [Module 2: Implement and Manage Compute Resources (20-25%)](./2-MANAGE-COMPUTE/README.md)
* Create and manage virtual machines
* Manage containers
* Create and manage App Service resources
* Implement Azure Batch

### [Module 3: Manage Storage (15-20%)](./3-MANAGE-STORAGE/README.md)
* Manage storage accounts
* Manage data in Azure Storage
* Configure Azure Files and Azure Blob Storage
* Manage storage security

### [Module 4: Manage Networking Resources (20-25%)](./4-MANAGE-NETWORKING/README.md)
* Implement virtual networking
* Configure load balancing
* Manage network security
* Configure private and public IP addressing

### [Module 5: Monitor and Maintain Azure Resources (15-20%)](./5-MONITOR-MANAGE/README.md)
* Monitor Azure resources
* Implement backup and recovery
* Manage Azure updates
* Monitor costs and optimize resources
