# Module 2: Implement and Manage Compute Resources (20-25%)

This module covers Azure compute services, including virtual machines, containers, App Service, and Azure Batch. It represents one of the largest portions of the AZ-104 exam.

## 📖 Microsoft Official Documentation & Learn

### 1. Create and Manage Virtual Machines
* **Microsoft Learn Path**: [Create and manage virtual machines](https://learn.microsoft.com/en-us/training/modules/create-windows-virtual-machine-in-azure/)
* **Key Topics**:
  * Plan and deploy virtual machines
  * Configure VM sizing and performance
  * Manage VM disks and storage
  * Configure networking for VMs
  * Implement VM extensions and custom script extensions
  * Manage VM availability and scale sets

### 2. Manage Containers
* **Microsoft Learn Path**: [Manage containers in Azure](https://learn.microsoft.com/en-us/training/modules/run-docker-containers-aci/)
* **Key Topics**:
  * Create and manage container instances
  * Deploy containers using Azure Container Instances (ACI)
  * Configure container registries
  * Manage container images and repositories

### 3. Create and Manage App Service Resources
* **Microsoft Learn Path**: [Create and manage App Service resources](https://learn.microsoft.com/en-us/training/modules/host-a-web-app-with-azure-app-service/)
* **Key Topics**:
  * Create and configure App Service plans
  * Deploy web apps and mobile apps
  * Configure deployment slots
  * Manage App Service scaling and performance
  * Configure SSL/TLS certificates

### 4. Implement Azure Batch
* **Microsoft Learn Path**: [Implement Azure Batch](https://learn.microsoft.com/en-us/training/modules/run-parallel-tasks-in-azure-batch-by-using-the-azure-cli/)
* **Key Topics**:
  * Create and manage Batch accounts
  * Configure Batch pools
  * Submit and manage batch jobs
