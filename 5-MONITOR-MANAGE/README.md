# Module 5: Monitor and Maintain Azure Resources (15-20%)

This module covers monitoring, backup, recovery, and maintenance of Azure resources, including cost optimization and Azure updates.

## 📖 Microsoft Official Documentation & Learn

### 1. Monitor Azure Resources
* **Microsoft Learn Path**: [Monitor Azure resources](https://learn.microsoft.com/en-us/training/modules/monitor-azure-resources/)
* **Key Topics**:
  * Configure Azure Monitor
  * Configure Log Analytics
  * Create and manage alerts
  * Configure Application Insights
  * Analyze metrics and logs
  * Configure diagnostic settings

### 2. Implement Backup and Recovery
* **Microsoft Learn Path**: [Implement backup and recovery](https://learn.microsoft.com/en-us/training/modules/protect-virtual-machines-with-azure-backup/)
* **Key Topics**:
  * Configure Azure Backup
  * Create and manage backup policies
  * Perform backup and restore operations
  * Configure Azure Site Recovery
  * Implement disaster recovery

### 3. Manage Azure Updates
* **Microsoft Learn Path**: [Manage Azure updates](https://learn.microsoft.com/en-us/training/modules/manage-updates-azure-vms/)
* **Key Topics**:
  * Configure Update Management
  * Manage VM updates
  * Configure patch schedules
  * Implement compliance monitoring

### 4. Monitor Costs and Optimize Resources
* **Microsoft Learn Path**: [Monitor costs and optimize resources](https://learn.microsoft.com/en-us/training/modules/analyze-costs-create-budgets-azure-cost-management/)
* **Key Topics**:
  * Use Azure Cost Management
  * Create and manage budgets
  * Analyze cost trends
  * Use Azure Advisor for optimization
  * Configure cost alerts
