# Module 1: Manage Azure Identities and Governance (20-25%)

This module covers identity management and governance in Azure, including managing users, groups, roles, and implementing Azure Policy for organizational compliance.

## 📖 Microsoft Official Documentation & Learn

### 1. Manage Microsoft Entra Users and Groups
* **Microsoft Learn Path**: [Manage Microsoft Entra users and groups](https://learn.microsoft.com/en-us/training/modules/manage-azure-active-directory/)
* **Key Topics**:
  * Create and delete users
  * Create and manage groups
  * Manage user and group properties
  * Manage licenses in Microsoft Entra ID
  * Manage external users and B2B collaboration
  * Configure self-service password reset (SSPR)

### 2. Manage Access to Azure Resources
* **Microsoft Learn Path**: [Manage access to Azure resources](https://learn.microsoft.com/en-us/training/modules/manage-access-azure-resources/)
* **Key Topics**:
  * Understand Azure role-based access control (RBAC)
  * Manage built-in Azure roles
  * Assign roles at different scopes (subscription, resource group, resource)
  * Interpret and audit access assignments
  * Create and manage custom roles

### 3. Manage Azure Subscriptions and Governance
* **Microsoft Learn Path**: [Manage Azure subscriptions and governance](https://learn.microsoft.com/en-us/training/modules/manage-azure-subscriptions/)
* **Key Topics**:
  * Implement and manage Azure Policy
  * Configure resource locks (read-only and delete)
  * Apply and manage tags on resources
  * Manage resource groups
  * Manage subscriptions and billing
  * Manage costs using alerts, budgets, and Azure Advisor
  * Configure management groups
