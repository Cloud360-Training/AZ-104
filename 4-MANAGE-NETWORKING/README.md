# Module 4: Manage Networking Resources (20-25%)

This module covers Azure networking services, including virtual networks, load balancing, network security, and IP addressing. It is one of the most important modules for AZ-104.

## 📖 Microsoft Official Documentation & Learn

### 1. Implement Virtual Networking
* **Microsoft Learn Path**: [Implement virtual networking](https://learn.microsoft.com/en-us/training/modules/design-implement-virtual-network/)
* **Key Topics**:
  * Create and configure virtual networks
  * Create and configure subnets
  * Configure network peering
  * Configure DNS settings
  * Configure network security groups (NSGs)
  * Implement Azure Firewall

### 2. Configure Load Balancing
* **Microsoft Learn Path**: [Configure load balancing](https://learn.microsoft.com/en-us/training/modules/load-balance-web-traffic-with-application-gateway/)
* **Key Topics**:
  * Configure Azure Load Balancer
  * Configure Application Gateway
  * Implement traffic management
  * Configure health probes and backend pools

### 3. Manage Network Security
* **Microsoft Learn Path**: [Manage network security](https://learn.microsoft.com/en-us/training/modules/secure-network-connectivity-azure/)
* **Key Topics**:
  * Configure network security groups
  * Configure Azure Firewall rules
  * Implement DDoS protection
  * Configure network access control lists (ACLs)

### 4. Configure Private and Public IP Addressing
* **Microsoft Learn Path**: [Configure IP addressing](https://learn.microsoft.com/en-us/training/modules/configure-private-public-ip-addressing/)
* **Key Topics**:
  * Configure static and dynamic IP addresses
  * Create and manage public IP addresses
  * Configure private IP addressing
  * Implement IP forwarding
